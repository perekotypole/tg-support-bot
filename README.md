# Information bot with a support manager connection (telegram) 
Bot that gives information to users and then forwards tham to the manager

Features:
- sending default messages by request
- connect a user to a support manager via the bot (a intermediary profile creates a group chat with a manager and the bot, the bot forwards a user's messages to the chat and sends the manager's messages to the user)
- multilingual
- ability to make live configuration changes
- re-authorisation without stopping execution
